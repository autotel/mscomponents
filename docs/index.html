Slider

Slider produces a vertical or horizontal slider that allows to control a value from dragging with the mouse.

new Slider(parent: jquery, options: object)
Parameters
parent (jquery) or DOM element to which this slider will be attached. defaults to $("body")
options (object) object containing options
Name Description
options.css String additional css properties for the slider
options.valueFunction function
options.label object :""
options.valueFunction object :function(val){ return val; }
options.value object :0
options.data object :{value:0}
options.ertical object :true defines the operation to apply to the internal value upon evaluation. the default is just linear
Example
mySlider new MsComponents.Slider($("body"),{vertical:false,value:0.73});
Instance Members
▸ setData(to, emit = false)

Set the data to a value, and perform the graphic changes and data bindings that correspond to this change. If you wanted to change the value, but not get this change reflected in the slider position, you would assign the slider.data.value to your value.

setData(to: number, emit: boolean?): undefined
Parameters
to (number) target value
emit (boolean? = false) not ready wether to emit through syncman
Returns
undefined: no return
Example
mySlider.setData(0.53);
